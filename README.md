# 💡 Arduino Light Sensor



## 📝 Description
This project explores the functionality of a photoresistor (LDR - Light Dependent Resistor) by building a simple light-sensing circuit using an Arduino Uno. The main goal of this project is to understand how a photoresistor reacts to changes in ambient light and how a microcontroller can read these analog values. By converting the light intensity into electrical signals, the Arduino can control an output, such as automatically turning on an LED when the environment gets dark. It is a great hands-on introduction to working with analog sensors.

## 🛠️ Languages and Utilities Used
- **Arduino IDE** (Integrated Development Environment)
- **C/C++** (Arduino Programming Language)
- **OrCAD PSpice** (Circuit Simulation)

## 🖥️ Environments Used
- **Windows 11**

## 🧰 Hardware Components Used
- **1x** Arduino Uno Board
- **1x** Breadboard
- **1x** Photoresistor (LDR)
- **1x** LED
- **1x** 220Ω Resistor (for the LED)
- **1x** 10kΩ Resistor (for the photoresistor voltage divider)
- Male-to-Male Jumper Wires

## 💻 OrCAD Simulation
Before wiring the physical components, the circuit was designed and simulated using OrCAD. This step ensures that the voltage divider logic works correctly and provides safe and readable voltage levels for the Arduino's analog input.
